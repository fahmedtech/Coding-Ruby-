IT 211 -- Project 1
Compute the Monthly Payment for a Loan

 

  • Write a Ruby script that computes monthly interest payments for a home or car loan.
     
      Inputs: p = Loan amount (principal)
        r  = Interest rate in percent
        n = Number of years (duration of loan)
         
      Output: m = Monthly payment

    Here is the formula:
     
    m =
    p r / 1200.0
    1 - (1.0 + r / 1200.0) –12 n

  • Use prompts for input to let the user know what to input; use text to explain what the output means.
     
  • Round the monthly payment is to the nearest cent.
     
  • Use Cel2Fahr and Distance as examples.
     
  • Validate p, r, and n with if statements to insure that they are all positive.
     
  • Grading Breakdown: Functionality: 70%, Indentation: 10%, Source Code Comments: 10%, Submitted Correctly: 10%.
    Source code comments should the following:
      (1) a header at the top with your name, project number, and submission date, and
      (2) a brief explanation at the beginning of each section of what that section of the script does.
    Submitted correctly means that your name and project number are included in the filename of the submission.
