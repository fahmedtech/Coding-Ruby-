IT 211 -- Project 6

User Defined Classes

  • Deliverables: Souce code file for class; source code file that creates objects and tests the class; .yaml file that contains the array of objects in YAML format.
     
  • Goal:   Design, implement, and test a user defined class similar to the Person or SavingsAccount classes. You may also get some ideas from the BabyDragon class on Page 133 in the textbook.
     
  • Examples of Classes to Use for your Project: AirplanePassenger, Baby, CatalogItem, DriversLicense, GroceryItem, InsurancePolicy, LibraryBook, LibraryCD, LibraryPatron, Pet, Store, Student.
     
  • UML Diagram: Submit a UML diagram for your class. You can include the UML diagram as comments at the top of your class code (use typewriter graphics if you do this), or in a separate file.
     
  • Example Classes:   Do not use these classes that are posted as examples on the class website or discussed in the textbook: Person, SavingsAccount, Coin, Automobile, BabyDragon.
     
  • Design Specifications:   For full credit, your class should have
     
    1. An initialize method.
       
    2. At least four fields, including at least one read-only field and at least one read/write field.
       
    3. A to_s method.
       
    4. At least one method of your choice, in addition to the initialize, to_s methods and the getters and setters.

  • Code Specifications:   Put the class code and test code in separate files.
     
    1. Write explicit getter and setter methods.
       
    2. Include source code comments.
       
    3. Indent your source code properly. Each indentation should be 2 spaces.

  • Testing Specifications:
     
    1. Create an array that contains at least three objects.
       
    2. Call each method of your class to test it.
       
    3. Call the to_yaml method to convert the array of objects to a YAML string. You don't need to save the YAML string to an output file. To call use the to_yaml method, you need
        require 'yaml'
      at the top of your test script. If the name of your object array is objects, print the YAML string of the objects like this:
        print objects.to_yaml, "\n"

  • Grading Breakdown:   UML Diagram: 25%; Creativity/Originality: 15%; Code Functionality: 35%; Test Code Completeness: 25%.
